How to get the URL from your calendar

Apple Calendar (iPhone)

  1. Calendar app → tap Calendars at the bottom
  2. Tap the (i) next to a calendar
  3. Toggle Public Calendar on
  4. Tap Share Link, then Copy

Google Calendar

  1. calendar.google.com → gear → Settings
  2. Pick the calendar → Integrate calendar
  3. Copy Secret address in iCal format

Outlook / Microsoft 365

  1. outlook.office.com → Settings → Calendar
  2. Shared calendars → Publish a calendar
  3. Pick a calendar, choose “Can view all details”
  4. Copy the ICS link (not the HTML one)

Privacy: the URL is non-guessable. Nobody can find it without you sending it. I check it once a day, never write to your calendar, never store events — only the matches I email you about.
